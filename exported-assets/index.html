Semana 1: Recolección de Datos

En esta primera semana, nos enfocamos en obtener datos de irradiancia solar utilizando la API de PVGIS (Photovoltaic Geographical Information System) para Madrid, España.

Fuente de datos: PVGIS

PVGIS es una herramienta gratuita que proporciona datos de irradiancia solar y estimaciones de producción fotovoltaica para Europa, África y gran parte de Asia. Los datos se guardaron en formato CSV para su posterior análisis.

Parámetros utilizados

Parámetro Valor

Código de extracción

import requests
import pandas as pd

# Parámetros para Madrid
lat, lon = 40.4168, -3.7038
year = 2022

# URL de la API PVGIS
url = f"https://re.jrc.ec.europa.eu/api/seriescalc"
params = {
    'lat': lat,
    'lon': lon,
    'startyear': year,
    'endyear': year,
    'outputformat': 'csv',
    'components': 1,
    'browser': 0
}

# Solicitud de datos
response = requests.get(url, params=params)
data = pd.read_csv(StringIO(response.text), skiprows=10, nrows=8760)
data.to_csv('datos_irradiancia_madrid.csv', index=False)

Semana 3: Modelado Fotovoltaico

En la tercera semana implementamos modelos de sistemas fotovoltaicos utilizando la librería pvlib de Python para simular la producción de energía.

Librería pvlib ModelChain

Utilizamos la clase ModelChain de pvlib que integra todos los pasos necesarios para modelar un sistema fotovoltaico: desde la irradiancia en el plano hasta la potencia AC de salida.

Supuestos del modelo

  • Potencia nominal del sistema: 5 kWp
  • Orientación: Sur (azimut 180°)
  • Inclinación óptima: 35°
  • Eficiencia del inversor: 96%
  • Pérdidas del sistema: 14%
  • Tecnología: Silicio cristalino

Código de modelado

import pvlib
from pvlib import pvsystem, modelchain, location

# Definir localización
madrid = location.Location(40.4168, -3.7038, 'Europe/Madrid', 667, 'Madrid')

# Configurar sistema PV
sistema_pv = pvsystem.PVSystem(
    surface_tilt=35, 
    surface_azimuth=180,
    module_parameters={'pdc0': 300, 'gamma_pdc': -0.004},
    inverter_parameters={'pdc0': 5000, 'eta_inv_nom': 0.96}
)

# Crear modelo completo
mc = modelchain.ModelChain(sistema_pv, madrid, 
                          aoi_model='physical', 
                          spectral_model='no_loss')

Introducción a la Energía Solar

La energía solar fotovoltaica representa una de las tecnologías renovables más prometedoras del siglo XXI. Mediante el efecto fotovoltaico, las células solares convierten directamente la radiación solar en electricidad, ofreciendo una fuente de energía limpia, silenciosa y prácticamente inagotable.

Los sistemas fotovoltaicos han experimentado una dramática reducción de costos en las últimas décadas, convirtiéndose en la fuente de electricidad más económica en muchas regiones del mundo. Esta tecnología no solo contribuye a la reducción de emisiones de gases de efecto invernadero, sino que también ofrece independencia energética y beneficios económicos a largo plazo.

El funcionamiento de un sistema fotovoltaico depende de múltiples factores: la irradiancia solar disponible, la temperatura ambiente, la orientación e inclinación de los paneles, y las características técnicas de los componentes. Un análisis detallado de estos parámetros es fundamental para optimizar el diseño y predecir el rendimiento del sistema.

Este proyecto analiza datos reales de irradiancia solar para Madrid, utilizando herramientas como PVGIS y la librería pvlib de Python, con el objetivo de modelar y evaluar el comportamiento de un sistema fotovoltaico típico. Los resultados proporcionan información valiosa para la toma de decisiones en proyectos de energía solar.
